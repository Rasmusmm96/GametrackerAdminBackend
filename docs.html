GameTracker Admin API

POST /games/add/{title}/{developer}/{publisher}/{release_date}

Returns: A boolean, true if the game was created, false if it failed.

Inputs:

  • title: The title of the game to add.
  • developer: The developer of the game to add.
  • publisher: The publisher of the game to add.
  • release_date: The release date of the game to add.
PUT /games/update/{id}/{title}/{developer}/{publisher}/{release_date}

Returns: A boolean, true if the game was updated, false if it failed.

Inputs:

  • id: The id of the game to update.
  • title: The updated title.
  • developer: The updated developer.
  • publisher: The updated publisher.
  • release_date: The updated release date.
DELETE /games/delete/{id}

Returns: A boolean, true if the game was deleted, false if it failed.

Inputs:

  • id: The id of the game to delete.
